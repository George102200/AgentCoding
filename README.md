# Agentic Coding 網頁開發專案 🚀

這是一份深度學習與 Agentic Coding 課程的網頁開發作業，透過 AI 輔助逐步完成從基礎 HTML 到進階資料串接的個人靜態網站。

## 🌟 專案特色 (Features)
* **HW1 (基本套件):** 使用原生 HTML/CSS 建立基礎架構。
* **HW2 (進階套件):** 導入 Tailwind CSS 實現現代化響應式設計 (RWD)。
* **HW3 (動態互動):** 透過 JavaScript 實作深色模式切換功能。
* **HW4 (資料渲染):** 使用 Fetch API 非同步讀取 JSON 資料庫，動態生成網頁內容。
